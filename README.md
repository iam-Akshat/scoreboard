# Scoreboard API

> A Single Page App for a restaurant website using vanilla JS and Webpack.


## Built With
- JavaScript
- HTML 5 and CSS 3.



## Getting Started

- `git clone git@github.com:iam-Akshat/scoreboard.git`
- `cd scoreboard`
- optional for linters `npm i`
- Open index.html
